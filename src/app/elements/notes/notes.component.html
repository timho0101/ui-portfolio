Promises VS Observables
Promises:
- Prepresent a single value in the future
Observables:
- Pepresent O or more values now or in the future.
- Can be Synchronous.
- Improved Error Handling.
- Can be Closed Independently of Returning a Value.
- Can deal with time.
- Advanced operations.
- Mathmetical Aggregation
- Buffering
- Debounce
- Distinct
- Filtering
- Combining Observables
- Retry
- Nodemon: npm install nodemon.
